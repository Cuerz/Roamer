# Roamer

对[InfoSmse](https://github.com/Security-Magic-Weapon/InfoSmse)源码二次开发

可视化信息搜集工具，支持Hunter和Fofa语法查询。

# 使用方法

1.运行环境为java11+.
2.运行前请填写在jar包所在目录的config.yaml文件，里面是相关资产测绘平台的配置。

3.运行命令为

```
java -jar Roamer.jar
```
